## Pantheon OS — Autonomous Trading System

8 AI agents. One pipeline. No human in the loop.

```
Hermes (590+ live suppliers)
  → Icarus   signal watcher
  → Hades    compliance firewall (OFAC, EU sanctions, ESG)
  → Artemis  macro context (VIX, S&P regime, sector ETFs)
  → Pythia   pattern learning + Kelly position sizing
  → ZEUS     LLM reasoning (Claude Sonnet)
  → Ares     bracket order execution (Interactive Brokers)
  → Argus    portfolio monitor + drawdown kill switch
  + Apollo   daily research (arXiv, earnings, self-improvement loop)
```

**Live on Hetzner.** Paper trading. Agents level up from TRAINEE → DIRECTOR as they build a verified win rate. Live trading unlocks automatically when all agents clear Senior.

**Stack:** Python · Claude Sonnet 4.6 · ChromaDB · Supabase · Kafka · Redis · IB Gateway · Grafana · Docker · Hetzner
