## Architecture Summary

6-agent pipeline:
1. **Icarus** — fetches signals from Hermes API + Kafka
2. **Artemis** — market data (yfinance + FRED)
3. **Apollo** — research synthesis (arXiv + self-improvement)
4. **Hades** — compliance filter (OFAC, ESG, blocked tickers)
5. **Pythia** — position sizing (pattern-based confidence)
6. **Zeus** (director) — LLM reasoning + final decision
7. **Ares** — IB order execution (bracket orders)
8. **Argus** — portfolio monitor + kill switch (8% drawdown)

All agents report health to Supabase every 60s. Grafana reads live from Supabase Postgres.
